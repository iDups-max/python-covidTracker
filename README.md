# covidTracker

covidTracker is a small python script that shows covid-19 info in CLI + on a  unicorn pHAT.

## How to use.

Clone repo

```bash
git clone https://github.com/anton25360/python-covidTracker.git
```
Run ```sudo python main.py``` to start.
